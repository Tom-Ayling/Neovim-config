# Requirements
[Cmake 3.x](https://github.com/Kitware/CMake/releases/tag/v3.31.6), [ripgrep](https://github.com/BurntSushi/ripgrep#installation), [fd-find](https://github.com/sharkdp/fd#installation), [NerdFont](https://www.nerdfonts.com/) (JetBrainsMonoNL Nerd Font Mono), [Tree-Sitter cli](https://github.com/tree-sitter/tree-sitter/blob/master/crates/cli/README.md)

# Install
- Install the requirements
- Clone the repo into the neovim config dir (Appdata\Local\nvim on windows)
